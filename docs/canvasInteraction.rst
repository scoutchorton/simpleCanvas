Methods

``resize(w, h);``
###########################
 Sets the width and height in pixels of the canvas.

 +-----------------------------+
 |Parameters                   |
 +--------+--------------------+
 |Variable|Description         |
 +========+====================+
 |``w``   |New width of canvas |
 +--------+--------------------+
 |``h``   |New height of canvas|
 +--------+--------------------+
